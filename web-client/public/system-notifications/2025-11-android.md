# 安卓主机丐！

使用方法：

1. [下载该版本](https://1drv.ms/u/c/83c52b42e4822761/EcRdPTgOS1FEhau8cakphtwBXdw1DDY_jbsNwfxUUM058Q?e=ICiJNG)并安装 [SkyEmu](https://skyemu.app/) APK。
2. 将 [`virtual_rom.gba`](virtual_rom.gba) 放到手机 `Download` 目录下。
3. 使用 C-to-C 或其他 OTG 方式连接主机丐与手机。
4. 打开 SkyEmu，授予 USB 权限后选择 `virtual_rom.gba` 即可开始游戏。
